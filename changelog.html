Change Log

v0.7.4, Jan 30, 2019

  • Added the Fast selection feature in the Reader.
  • Removed support of 3D Touch in the Reader.
  • Added OCR function for DjVu and PDF pages without text layer.
  • When adding a bookmark, the title of the chapter is added to the text.
  • Removed default paragraph indent from EPub, Mobi, FB2 display styles.

v0.7.3, Jan 23, 2019

  • Fixed the issue with resetting playback speed in audiobooks after paused.
  • Fixed the issue with turning pages in landscape orientation using a hardware keyboard.
  • Few small improvements

v0.7.1, Jan 14, 2019

  • Czech localization.
  • Added Dashboard/Notes screen for viewing user-made marks in all books.
  • The Home button indicator on the iPhone X (iPad 11-inch) is always hidden while reading a book in full screen.
  • Fixed an issue with images in EPub,Mobi,Fb2 books, when the light theme is applied.
  • Few minor fixes and improvements.
  • Reduced the application size.

v0.7.0, Dec 06, 2019

  • Improved Tor Network support (tor 0.3.5.6-rc).
  • Fixed Tor Network reconnection issue.
  • Few small improvements.

v0.6.9, Dec 29, 2018

  • Added adaptation of book images in the palette of the current theme (applicable only for light themes).
  • Added hiding the Home button indicator on iPhoneX (iPad 11) when reading book in pagination mode.
  • Fixed the application crash due to insufficient memory when processing many very large ebooks.
  • Fixed back navigation in System Dictionaries dialog.
  • Fixed an issue where unable to share a MOBI book from another application to KyBook if the Kindle App is installed on the device.
  • Fixed some background crashes.

v0.6.8, Dec 23, 2018

  • On the book card in the bookshelf, the same series is now always displayed as in the parent section or the bookshelf.
  • Fixed issues with rendering margins of the wrong size in the EPub.
  • Fixed some issues with audio streaming.
  • Supports of streaming from One Drive storage.
  • Support of streaming PDF metadata from the Cloud Storage.
  • Added the ability to limit the maximum size of the cloud storage files cached on the device.
  • Few internal improvements and optimizations

v0.6.7, Dec 18, 2018

  • Added an option to automatically playback the speech synthesis when open EPub,MOBI,FB2 book (disabled by default, enabled via the TTS settings).
  • Changed default behavior to "Copying" instead of "Moving" when transfering files between/to/from Cloud Storage.
  • Optimized extraction of cover images from FB2 books.
  • Added support of the "Share" function for several books at once from a bookshelf (located on a device) as a zip-archive.
  • Added the File Manager widget in the advanced interface mode.
  • A few minor improvements and bug fixes.

v0.6.6, Dec 14, 2018

  • Optimized the page spread for PDF/DjVu/Comics in page curl mode.
  • Added an animation of adding/deleting note markers.
  • PDF Reflow mode now is aware of a page's links.
  • Fixed wrong layout of a search bar in a bookshelf and a catalog's page on iPhone X.
  • Improved a text selection in the Reader for EPub/Mobi/Fb2 books.
  • Added an ability to fold/unfold the table of contents in the book.
  • Improved fast forward/rewind for the speech synthesis. In the EPub/Mobi/Fb2 books, forward/rewind now takes into account the paragraph or sentence boundary. In PDF/DjVu books, forward/rewind starts playback from the next/previous page, accordingly.
  • A few minor improvements and bug fixes.

v0.6.5, Dec 8, 2018

  • Fixed an issue with selection markers which may leave off after page changes.
  • Fixed an issue with opening a book through the iOS Files App from the KyBook's folders in the iCloud or on the device.
  • Fixed a rare error when it is impossible to download a book, a dictionary or something else.
  • Rendering of PDF now takes into account the preset CropBox of page.
  • Supports of the page spread mode in the landscape orientation on iPad for PDF/DjVu/Comics.
  • Added an option 'No animation' for page changes.
  • Optimized the full screen toggling for PDF/DjVu/Comics.
  • Updated the Chinese localization.
  • Updated the Spanish localization.
  • A few minor improvements and bug fixes.

v0.6.3, Nov 29, 2018

  • Added the "Exclude" context menu item to "Recent" and "Reading" bookshelves.
  • Added the new sorting mode "by Format" in the bookshelf.
  • Added the Chinese localisation. Many thanks to 陈寿山 (ShaneChen) for the translation.
  • Few bug fixes and minor improvements.

v0.6.2, Nov 27, 2018

  • Faster loading of large plain text books.
  • Added the setting "Background image is a pattern" for ebooks in the EPUB, MOBI, FB2 formats. This setting allows to use a background image as a background for the header and footer in the base interface mode. Disabled by default, may be enabled through an ebook settings.
  • New experimental scrolling mode for ebooks in the EPUB, MOBI, FB2 formats. Prevents flickering and reduces CPU usage and an energy consumption. Especially good when the text reader has a background image. Disabled by default, may be enabled through the reader settings.
  • Fixed an issue with occasionally hidden navigation bar on "Library" and "Catalogs" screen in the base interface mode.
  • Added the preferences profile feature. Now an user can separately set preferences for any ebook in the EPUB, MOBI, FB2 formats.
  • Added a message and alert to a bookshelf about "a book is still processing" when an user tries to interact with not yet loaded book.
  • Fixed an issue when the userstyle in an ebook preferences was ignored.
  • Various improvements and fixes.

v0.6.1, Nov 20, 2018

  • Improved stability and performance of audio book streaming from the cloud storage.
  • Improved stability and performance during obtaining metadata of audio book from the cloud storage.
  • Fixed the crash during opening some EPUBs.
  • Fixed the base stylesheet used for EPUB and MOBI ebooks.
  • Fixed an issue when unable delete a non-existent ebook file through the Bookcard screen.
  • Added the ability to automatically create a playlist of mp3 files in the folder.
  • Added the Spanish localization. Many thanks to José Labán for the translation.
  • Added the Google Translator service.
  • Added the ability to delete a book record from the database without deleting the book files through the Bookcard / Misc screen.
  • Various minor improvements and bug fixes.

v0.6.0, Nov 2, 2018

  • Fixed a black background of images in some pdf.
  • Fixed a wrong layout in Reader after a device unlock if device orientation was changed.
  • Fixed some issues of the files synchronization.

v0.5.9, Oct 31, 2018

  • Fixed an issue with the layout when toggling fullscreen mode in Reader with pagination on iOS 12.

v0.5.8, Oct 28, 2018

  • Added an ability to view a web page of a book's author in the cataloguer bookshelf.
  • Various bug fixes and stability improvements.
  • Added an ability to view a list of books uploaded to KyBook cloud and free up space by deleting them.
  • Fixed an issue with the status bar in a fullscreen mode when using the advanced interface on iOS 12.
  • Now the status bar on iPhone X(S, R) in hidden in a fullscreen by default. To return to the previous behaviour opt in "iPhone X Notch" in the Reader settings.
  • Fixed a sort order in the Photos picker.
  • Removed the delay in calculating the remaining pages in the current chapter.
  • Improved synchronization of edited or deleted the cataloguer's categories (seria, author, publisher, subject)

v0.5.6, Oct 15, 2018

  • Trash Bin for deleted books.
  • Feature to manually to add a download link.
  • Fixed crashes when opening some EPubs.
  • Various bug fixes and stability improvements.

v0.5.5, Oct 2, 2018

  • Reader rotation lock (available on iPhone only).
  • Reader draws a full-sized hyphen in EPUB,MOBI books when the readability setting Hanging punctuation is disabled.
  • Added an ability to batch download from OPDS Catalog (requires Pro).
  • Added an ability to select multiple books in the Catalogue categories "New", "Read", "Unread", etc.
  • Fixed the crash during selection a sentence spread on multiple pages in PDF/DJVU books.
  • Fixed an issue with the reader header/footer in an advanced interface and a fullscreen mode.
  • Various minor improvements and bug fixes.

v0.5.4, Sep 25, 2018

  • Optimized the work of the bookcovers cache.
  • Improved tab switching in the advanced mode of user interface.
  • Implemented the restoring of the pinned tab in the advanced mode of the user interface after the application restart.
  • Added the ability to batch editing of the books metadata (requires Pro).
  • Added "All Books" category in the Cataloguer.
  • Added support for Caliber Content Server Tags/Series info when downloading a book from OPDS catalog.
  • Added filtering of child shelves when searching in the bookshelf.
  • Fixed an issue with an repeating alert 'Mock filestore' for the shared folder (SMB/CIFS).

v0.5.3, Sep 20, 2018

  • Added an ability to pick Iowan Old Style Font.
  • Fixed bad layout on iPad 12.9 in "Add WebDAV/Samba/OPDS" Dialogs.
  • Added display a full path to deleted files in "Confir Delete" Dialog.
  • Implemented a more seamless interaction with cloud storage.
  • Increased the maximum values of the text margin in the Reader.
  • Added support for an external keyboard in the Reader.
  • Added an ability to change/set the password in the KyBook Account.
  • Added display of a generated cover in the book cell when a book does not have a cover.
  • Fixed bad line breaks (in hyphenated text) on contractions (apostrophes).
  • Improved the English localization (Many thanks to github.com/karlic for the help).
  • Few bug fixes and minor improvements.
