История изменений

v0.7.0 - 06 Января 2019

  • Улучшена поддержка Tor Network (tor 0.3.5.6-rc).
  • Исправлена проблема переподключения к Tor Network.
  • Несколько небольших улучшений.

v0.6.9, 29 Декабря 2018

  • Добавлена адаптация книжных изображений в палитре текущей темы (применимо только для светлых тем).
  • Добавлено скрытие индикатора кнопки "Домой" на iPhone X (iPad 11) при чтении книги в режиме с разбиением на страницы.
  • Исправлено аварийное завершение работы приложения из-за недостатка памяти при обработке множества очень больших электронных книг.
  • Исправлена обратная навигация в диалоге системных словарей.
  • Исправлена проблема, когда невозможно было отправить MOBI книгу из другого приложения в KyBook, если на устройстве установлено приложение Kindle.
  • Исправлены некоторые фоновые сбои.

v0.6.8, 23 Декабря 2018

  • На книжной карточке книжной полки теперь всегда отображается та же серия, что и в родительском разделе или на книжной полке.
  • Исправлены проблемы с рендерингом полей неправильного размера в EPub.
  • Исправлены некоторые проблемы с потоковым аудио.
  • Поддержка потоковой передачи из хранилища One Drive.
  • Поддержка потоковой передачи метаданных PDF из облачного хранилища.
  • Добавлена возможность ограничения максимального размера файлов облачного хранилища, кэшируемых на устройстве.
  • Несколько внутренних улучшений и оптимизаций.

v0.6.7, 18 Декабря 2018

  • Добавлена опция автоматического запуска синтеза речи при открытии EPub, MOBI, FB2 книги (отключено по умолчанию, включается через настройки TTS).
  • Изменено поведение по умолчанию на «Копирование» вместо «Перемещение» при передаче файлов между/в/из облачного хранилища.
  • Оптимизировано извлечение изображений обложек из FB2 книг.
  • Добавлена поддержка функции «Поделиться» для нескольких книг одновременно с книжной полки (расположенной на устройстве) в виде zip-архива.
  • Добавлен виджет файлового менеджера в расширенном режиме интерфейса.
  • Несколько незначительных улучшений и исправлений ошибок.

v0.6.6, 14 Декабря 2018

  • Оптимизирован разворот страницы для PDF/DjVu/Comics в режиме перелистывания страниц.
  • Добавлена анимация добавления/удаления маркеров заметок.
  • Режим PDF Reflow теперь поддерживает ссылки на странице.
  • Исправлено неправильное расположение панели поиска на книжной полке и странице каталога на iPhone X.
  • Улучшен выбор текста в читалке для EPub/Mobi/Fb2 книг.
  • Добавлена возможность свернуть/развернуть оглавление в книге.
  • Улучшена перемотка вперед/назад для синтеза речи. В EPub/Mobi/Fb2 книгах перемотка вперед/назад теперь учитывает границу абзаца или предложения. В PDF/DjVu книгах перемотка вперед/назад начинает воспроизведение со следующей/предыдущей страницы соответственно.
  • Несколько незначительных улучшений и исправлений ошибок.

v0.6.5, 8 Декабря 2018

  • Исправлена проблема с маркерами выделения которые могли остаться после смены страницы.
  • Исправлена проблема с открытием книги через приложение iOS Files из папок KyBook в iCloud или на устройстве.
  • Исправлена редкая ошибка, при которой невозможно было загрузить книгу, словарь или что-то еще.
  • Рендеринг PDF сейчас учитывает заданный CropBox страницы.
  • Поддержка режима разворота страницы в альбомной ориентации на iPad для PDF/DjVu/Comics.
  • Добавлена опция «Без анимации» для смены страницы.
  • Оптимизировано полноэкранное переключение для PDF/DjVu/Comics.
  • Обновлена китайская локализация.
  • Обновлена испанская локализация.
  • Несколько незначительных улучшений и исправлений ошибок.

v0.6.3, 29 Ноября 2018

  • Добавлен пункт контекстного меню «Исключить» в книжные полки «Недавнее» и «Читаются».
  • Добавлен новый режим сортировки «по формату» в книжную полку.
  • Добавлена китайская локализация. Большое спасибо 陈寿山 (ShaneChen) за перевод.
  • Несколько исправлений ошибок и незначительных улучшений.

v0.6.2, 27 Ноября 2018

  • Быстрая загрузка больших текстовых книг.
  • Добавлен параметр «Фоновое изображение - узор» для книг в формате EPUB,MOBI,FB2. Этот параметр позволяет использовать фоновое изображение в качестве фона для верхнего и нижнего колонтитулов в режиме базового интерфейса. Отключено по умолчанию, включается в настройках книги.
  • Новый экспериментальный режим прокрутки для книг в формате EPUB,MOBI,FB2. Предотвращает мерцание и снижает потребление ЦП и потребление энергии. Режим особенно хорош, когда у читалки включено фоновое изображение. Отключено по умолчанию, включается в настройках читалки.
  • Исправлена проблема со случайно скрытой навигационной панелью на экране «Библиотека» и «Каталоги» в режиме базового интерфейса.
  • Добавлена функция профиля настроек. Теперь пользователь может отдельно устанавливать предпочтения для каждой книги в формате EPUB,MOBI,FB2.
  • Добавлено сообщение и предупреждение книжной полки о том, что «книга все еще обрабатывается», когда пользователь пытается взаимодействовать с еще не загруженной книгой.
  • Исправлена ошибка, когда пользовательский стиль в настройках книги игнорировался.
  • Различные улучшения и исправления ошибок.

v0.6.1, 20 Ноября 2018

  • Улучшена стабильность и производительность потоковой передачи аудио книг из облачного хранилища.
  • Улучшена стабильность и производительность при получении метаданных аудио книг из облачного хранилища.
  • Исправлен сбой при открытии некоторых EPUB.
  • Исправлена базовая таблица стилей применяемая для EPUB и MOBI книг.
  • Исправлена ошибка, при которой невозможно удалить несуществующий файл через карточку книги.
  • Добавлена возможность автоматического создания плейлиста из mp3-файлов в папке.
  • Добавлена испанская локализация. Большое спасибо José Labán за перевод.
  • Добавлен сервис перевода Google Translator.
  • Добавлена возможность удалить книжную запись из базы данных без удаления файлов книги через экран Книжная карточка / Разное.
  • Различные незначительные улучшения и исправления ошибок.
